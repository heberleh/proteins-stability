Essa pasta contém programas para avaliação do conjunto
de dados.

É possível filtrar proteínas/variáveis por meio de um
arquivo externo contendo a lista de proteínas e um valor
de N que pode ser usado para cortar essa lista
(no caso de ranques).

O programa executa cros-validação Leave-one-out mas
é preciso ajustar o K para o número de amostras que vai 
executar.

Usei o código para testar 28 amostras e 22 amostras,
4 proteínas inseridas e todas as proteínas, assim
como um conjunto de teste independente com 6 amostras,
22+6 = 28.
